سياسة الخصوصية لشركة Black Box

تلتزم شركة Black Box بحماية خصوصية مستخدميها وتوضيح كيفية جمع واستخدام البيانات عبر منصاتها (AI Studio, Bx Wallet, Verification Sector).

1. البيانات التي نجمعها

  • بيانات الهوية: عند تسجيل الدخول عبر Google، نجمع (الاسم، البريد الإلكتروني، وصورة البروفايل).
  • بيانات المحفظة: سجلات شحن واستخدام عملة "Bx" لضمان حقوقك المالية.
  • بيانات التوثيق: الملفات والنصوص التي ترفعها لغرض التوثيق الرقمي وحماية الملكية.

2. كيف نستخدم بياناتك

نستخدم البيانات لتوفير الخدمات، ومعالجة المعاملات المالية، وإصدار شهادات التوثيق الرقمية، وتحسين تجربة المستخدم عبر الذكاء الاصطناعي.

3. حماية البيانات

نستخدم تقنيات تشفير الهاش (SHA-256) وسجلات غير قابلة للتعديل لحماية أصولك الموثقة ومعاملاتك المالية.

4. مشاركة البيانات

لا نقوم ببيع أو تأجير بياناتك لأي أطراف ثالثة. يتم مشاركة البيانات فقط مع بوابات الدفع المعتمدة لإتمام العمليات المالية.
